## for循环语句
``` java
for (int i = 0; i < 10; i++) {  
    System.out.println("hello world");  
}
```

## while语句
``` java
int i = 1;  
while (i < 10) {  
    System.out.println("hello world");  
    i++;  
}
```
## do{ } while语句
``` java
int i = 11;  
do {  
	//先执行再判断
    System.out.println(i);  
    i++;  
} while (i < 10);
```
